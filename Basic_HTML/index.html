Mein kurzer Aufsatz:

Linux ist heutzutage leichter zu nutzen als je zuvor. Es gibt zahlreiche Variationen und Derivate von Debian, Arch, RedHat und co., sodass für jeden Geschmack etwas dabei ist. Anfängern wird meist Ubuntu, ein Derivat von Debian, empfohlen, da es eine große Community hat, leichter als Windows zu bedienen ist und von Canonical gepflegt wird.

Für fortgeschrittenere User ist zum Beispiel Arch Linux etwas. Es verfolgt eine andere Philosphie von Betriebssystem als Debian/Ubuntu, was sich am deutlichsten in der Nutzung des Rolling-Release zeigt.
